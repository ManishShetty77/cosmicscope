🚀 CosmicScope — Space Exploration Portal

📌 Overview

CosmicScope is a full-stack web application that helps users explore space and astronomy in a simple and interactive way. It works completely offline and presents scientific data like asteroids, exoplanets, and space weather in an easy-to-understand format.

The project is built using React for the frontend, Node.js with Express for the backend, and SQLite as the database.

---

🎯 Features

👤 User Features

- User registration and login
- Save favorite asteroids and exoplanets
- Personal dashboard to view saved items
- Search functionality for quick data access
- Offline chatbot (AstroGuide) for astronomy queries

🛠️ Admin Features

- Secure admin login
- Add, edit, and delete asteroids
- Add, edit, and delete space weather data
- Changes update instantly in the application

🌌 Modules Included

- Astronomy Image Gallery
- Mars Rover Explorer
- Asteroid Dashboard
- Exoplanet Explorer
- Space Weather Monitor
- Space Timeline
- Space Glossary

---

🧠 AstroGuide Chatbot

AstroGuide is a built-in chatbot that works without internet.
It answers astronomy questions using the local database.

---

🏗️ Tech Stack

- Frontend: React + Tailwind CSS
- Backend: Node.js + Express
- Database: SQLite
- Authentication: Session-based + bcrypt

---

⚙️ How to Run

First Time Setup

npm install
npm run build
cd backend-template
copy .env.example .env
npm install
npm start

After First Time

cd backend-template
npm start

Open in browser:

http://localhost:3001
